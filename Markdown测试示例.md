## 数据验证实现方案

### JavaScript 实现

\```javascript
// 数据验证函数
function validateGSPData(data) {
  if (!data || typeof data !== 'object') {
    throw new Error('数据格式错误')
  }
  
  // 验证必填字段
  const requiredFields = ['id', 'name', 'date']
  for (const field of requiredFields) {
    if (!data[field]) {
      console.error(`缺少必填字段: ${field}`)
      return false
    }
  }
  
  return true
}

// 使用示例
const result = validateGSPData({
  id: '001',
  name: '质量检查记录',
  date: '2024-01-01'
})
\```

### Python 实现

\```python
def validate_gsp_data(data):
    """
    验证 GSP 数据的完整性
    """
    if not isinstance(data, dict):
        raise TypeError("数据必须是字典类型")
    
    required_fields = ['id', 'name', 'date']
    for field in required_fields:
        if field not in data:
            print(f"缺少必填字段: {field}")
            return False
    
    return True

# 使用示例
result = validate_gsp_data({
    'id': '001',
    'name': '质量检查记录',
    'date': '2024-01-01'
})
\```

### SQL 查询示例

\```sql
-- 查询最近 30 天的质量检查记录

  id,
  check_date,
  inspector,
  result,
  COUNT(*) as total
FROM quality_checks
WHERE check_date >= DATE_SUB(NOW(), INTERVAL 30 DAY)
GROUP BY inspector
ORDER BY check_date DESC;

## 🐛 如果出现问题

### 样式不生效

检查浏览器控制台是否有 CSS 加载错误

### 代码不高亮

1. 确认代码块指定了语言（如 \`\`\`javascript）
2. 检查 `highlight.css` 是否正确加载

### 渲染为纯文本

1. 确认组件中使用了 `v-html="renderMarkdown(content)"`
2. 检查是否正确导入了 `renderMarkdown` 函数
